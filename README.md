# Kay´s Snake Quest

**Kay´s Snake Quest** is a retro arcade Snake game inspired by 90s arcade cabinets, neon pixel art, CRT effects, fruit collecting, score chasing, and Spanglish arcade energy.

## Play Style

This is an arcade-themed Snake game with:

- Neon/pixel-art inspired visuals generated by code
- Spanglish interface
- Score and high score
- Levels and progression
- Custom visual elements
- Retro sound effects and music
- Keyboard controls
- Browser-playable version using Pygbag

## Controls

| Action | Key |
|---|---|
| Start / Restart | Space |
| Move | Arrow Keys |
| Move Alternative | W A S D |
| Pause | P |
| Quit Local Game | Esc or window close |

## Local Development

This project runs locally with Python and pygame-ce.

### Requirements

- Python 3.14+
- pygame-ce
- pygbag

### Install dependencies

```bash
python -m pip install -r requirements.txt

Run locally
python main.py
Web Version

The browser version is built using Pygbag.

pygbag .
